# Web Sentry

[English Version](./README.md)

## 简介

[Web Sentry](./) 是一个开源的Web服务。在选中网页的部分区域后并建立任务后，当该部分网页被改变时，用户会即时收到提醒。

https://github.com/websentry/WebSentry-Slave
