# 贡献指南


## 报告 Bug

我们使用 GitHub Issues。
所有问题应在 [主代码库](https://github.com/websentry/WebSentry/issues) 中报告。  

#### 安全漏洞

如果您发现了安全相关的漏洞，请先 [联系我们](https://github.com/websentry/WebSentry/blob/master/README.cn.md#%E8%81%94%E7%B3%BB%E6%96%B9%E5%BC%8F)。

## 提交建议

类似于“报告 Bug”。
